# ✨솔룩스 웹 프로젝트 '우리이웃'✨
## 💻프론트엔드 코드

'우리 이웃'은 주민들 간의 소통을 위한 아파트 커뮤니티 입니다. 

## 1. 기술 스택
- 프론트엔드: 리액트, html, js, css
- 백엔드: Spring
